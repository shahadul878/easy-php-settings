# WordPress.org Plugin Assets

Place your WordPress plugin directory submission assets here.
These files are used on the WordPress.org plugin page and are NOT included in the plugin ZIP.

## Required Assets

### Banner Images
- `banner-772x250.png` — Low-res banner (required)
- `banner-1544x500.png` — High-res (retina) banner (recommended)

### Icon Images
- `icon-128x128.png` — Low-res icon (required)
- `icon-256x256.png` — High-res (retina) icon (recommended)
- `icon.svg` — SVG icon (optional, preferred over PNG)

### Screenshots
- `screenshot-1.png` — Matches `readme.txt` screenshot descriptions
- `screenshot-2.png`
- `screenshot-3.png`
- (add more as needed)

## Notes
- File names are case-sensitive.
- PNG and JPEG formats are supported for banners and icons.
- Screenshots should be numbered sequentially starting from 1.
- The `.wordpress-org` folder is typically excluded from the plugin ZIP via `.distignore`.
